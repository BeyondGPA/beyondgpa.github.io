About BeyondGPA

BeyondGPA is an interactive data storytelling project that challenges the assumption that academic excellence alone defines early career success. Through a series of visual explorations, the project reveals how GPA, soft skills, and demographics all play nuanced roles in shaping professional outcomes.

Course Context

This project was created as the cumulative deliverable for the course INF8808 – Data Visualization during the Winter 2025 semester at Polytechnique Montréal. The objective was to apply the principles of effective data visualization using D3.js in a real-world style web application.

Tools & Technologies

The project was developed using Angular for component-driven architecture and D3.js for rich, responsive, and animated visualizations. The styling is fully responsive and transitions between light/dark modes using custom CSS variables and layout-aware theming.
